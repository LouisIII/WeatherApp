README

Weather app is a simple API based weather app user can use to check their current weather and search for the current weather an any global city.

USER EXPERIENCE
When the app loads users are prompted to type a city name into the search bar, if they want the weather in their current city there is a "current location" button located in the keyboard toolbar.

Upon searching the city all relevant current weather data will be displayed

APPLICATION CODE
The application properties and methods are managed by the LocationManager and Weather Manager classes. These classes are placed in the app environment and used in the various views.

Weather manager parses the JSON data from openweatherAPI
Location Manager gets the users location
